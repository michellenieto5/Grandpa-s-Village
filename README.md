# Grandpa-s-Village

Grandpa’s Village is a community-focused business network inspired by the idea that “it takes a village.” The goal of the platform is to help local businesses support one another while giving users a central place to discover trusted businesses within their community.

This project was built as a full-stack learning experience focused on cloud deployment, backend engineering, and RESTful API development.

## Thanks to
OSU CS 493 - Professor Nauman Chaudhry for your great work teaching hard things!

## What I learned

* Building and designing RESTful APIs using Flask
* Performing CRUD operations with cloud-hosted databases
* Deploying a web application using Google Cloud Platform (GCP)
* Connecting frontend and backend systems using JavaScript API calls
* Structuring relational data for businesses and reviews
* Working with HTML, CSS, and JavaScript for frontend development

## Current Features

* Business directory system
* Review system for businesses
* CRUD operations for businesses and reviews
* Cloud-hosted backend deployment
* Frontend connected to backend API endpoints
